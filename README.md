## Website Performance Optimization portfolio project

Optimize this online portfolio for speed! In particular, optimize the critical rendering path and make this page render as quickly as possible by applying the techniques learned in the [Critical Rendering Path course](https://www.udacity.com/course/ud884).

### Getting started

1. Check out the repository
1. Run a local server

  ```bash
  $> cd /mobile-portfolio/dist
  $> python -m SimpleHTTPServer 8080
  ```

1. Open a browser and visit localhost:8080
1. Download and install [ngrok](https://ngrok.com/) to the top-level of your project directory to make your local server accessible remotely.

  ``` bash
  $> cd /mobile-portfolio
  $> ./ngrok http 8080
  ```

#### Part 1: Optimize PageSpeed Insights score for index.html
- Copy the public URL `ngrok` gives you and run it through [PageSpeed Insights](https://developers.google.com/speed/pagespeed/insights/)! 

#### Part 2: Optimize Frames per Second in pizza.html

- Optimize `views/pizza.html` by modifying  `views/js/main.js`. Frames per second rate should result in 60 fps or higher.
